How to publish this on GitHub Pages (instructor only)

Go live in 3 minutes

1

Go to github.com → New repository → name it bootcamp-projects → Public → Create repository.

2

Click Add file → Upload files and drop all 5 files: index.html + the 4 PDF briefs.

3

Go to Settings → Pages → Branch: main → / (root) → Save. Wait about 60 seconds.

4

Your live link: https://YOUR-USERNAME.github.io/bootcamp-projects — paste it in the class WhatsApp group.
